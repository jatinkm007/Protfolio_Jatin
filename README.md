## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/jatinkm007/Protfolio_Jatin.git
```

Navigate to the project

```bash
cd Protfolio_Jatin
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser at

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
